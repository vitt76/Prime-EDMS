Organizations
^^^^^^^^^^^^^

An bug was fixed where an empty ``ORGANIZATIONS_URL_BASE_PATH`` value
would cause the work ``None`` to be appended to the URL after login.


Templating
^^^^^^^^^^

ignored was fixed.
