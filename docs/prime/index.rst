Инструменты разработки
----------------------

Для создания расширений используйте:

* Ручное создание структуры файлов
* Изучение существующих расширений (``converter_pipeline_extension``)
* Следование Django best practices
* Тестирование на локальной установке


Обзор
-----

Prime-EDMS построена на базе Mayan EDMS и поддерживает модульную архитектуру расширений (Django apps). Каждое расширение добавляет новую функциональность в систему.

Основные возможности расширений:

* Добавление новых моделей данных
* Создание пользовательских интерфейсов
* Интеграция с внешними системами
* Расширение API
* Кастомизация бизнес-логики


Быстрый старт
-------------

Для быстрого создания первого расширения:

1. Используйте скрипт ``create_extension.sh`` для генерации базовой структуры
2. Настройте модели, представления и шаблоны
3. Добавьте расширение в ``config.yml``
4. Пересоберите Docker образ

Подробные инструкции в :doc:`quick_start`.


Разработка
----------

Полное руководство по разработке расширений включает:

* Архитектуру и структуру кода
* Работа с моделями и базами данных
* Создание пользовательских интерфейсов
* REST API и сериализация
* Система разрешений
* Локализация и переводы
* Тестирование и отладка

См. :doc:`development_guide`.


Устранение неполадок
-------------------

Часто возникающие проблемы и их решения:

* Расширение не загружается
* Меню не отображается
* Переводы не работают
* Ошибки URL и маршрутизации
* Проблемы с Docker сборкой

Диагностика и решения в :doc:`troubleshooting`.


Инструменты
-----------

* ``create_extension.sh`` - скрипт для генерации структуры расширения
* ``extension_template.py`` - шаблон кода расширения
* Отладочные скрипты в ``contrib/scripts/``


Примеры
-------

В проекте доступны примеры реализаций:

* ``converter_pipeline_extension`` - расширенная система конвертации файлов


Поддержка
---------

При возникновении проблем:

1. Проверьте логи: ``docker logs prime-edms_app_1``
2. Изучите документацию и примеры
3. Создайте issue в репозитории проекта

Удачи в разработке расширений!
